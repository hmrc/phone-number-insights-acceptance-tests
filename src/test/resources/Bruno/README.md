
# Phone Number Insights API Bruno Collection

These are the API calls for manual testing in Bruno.

## How to import/open locally

1. Open Bruno
2. Click the + in the Collections tab
3. Select Open collection
4. Navigate to /src/test/resources/Bruno and click the Bruno folder once
5. Click Open

### How to update collection

Bruno is configured to your local acceptance test repository. 
Every time you make a change it will amend the collection automatically and update the changes in the repository.
Commit changes if you want to keep them, otherwise remove the Bruno changes before committing other changes.
